Class Overview

Welcome to the SITES Front End Development course! The primary goal of this course is to guide students through the basic knowledge required in the practice of front-end web development. Front-end web development is the creation of the graphical user interface (GUI) of a website so users can view and interact with that website. This is most achieved through the use of Hypertext Markup Language (HTML), Cascading Style Sheets (CSS), and JavaScript (JS), all of which students will learn in the SITES Front End Dev course.
